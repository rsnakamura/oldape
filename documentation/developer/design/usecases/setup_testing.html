Setup Session¶

Setup Session is an included use case within the Test Throughput case.

Related Requirements¶

  • A.1. System interprets Configuration Files.
  • A.2. System is invoked via the command line.

Precondition¶

The APE has setup a well-formed configuration file

Postcondition¶

The system is ready to start a Test

Failed Condition¶

The system has exited without starting a test.

Trigger¶

The All-In-One Performance Evaluation Tools (APETools) has executed the tot run subcommand.

Main Path¶

  1. The System converts the config file to runtime parameters.
  2. The System builds runtime components based on the parameters.

1.1. The System detects a misconfigured or missing required option.

1.1.1. The System raises a ConfigurationError

1.1.2. The System exits.
