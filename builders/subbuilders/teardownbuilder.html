Teardown Builder¶

A module to build tear-downs run at the end of a test set.

The TearDown Builder¶

TearDownBuilder(configfilename, storage[, ...]) A basic tear-down builder that just copies log and config files.
TearDownBuilder.configcopier
return:A file copier aimed at the config file
TearDownBuilder.logcopier
return:A file copier aimed at the log file
TearDownBuilder.teardown
return:A teardown object for the test-operator to run to cleanup
