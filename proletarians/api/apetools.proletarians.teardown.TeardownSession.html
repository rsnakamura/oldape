apetools.proletarians.teardown.TeardownSession¶

class apetools.proletarians.teardown.TeardownSession(tools, *args, **kwargs)[source]¶

The TearDown does whatever needs to be done after the test is completed.

It iterates over the tools passed in to the constructor. Each tool’s run method is called. Choosing tools and ordering them defines the TeardownSession algorithm.

Param:
  • tools: a list of tools to run
