The Naxxx MainÂ¶

A Module to hold a command-line version of the NAxxx.

set_logger(args) Creates a logger and sets the level based on args.
enable_debugging(args) Checks if pdb pudb has been set runs pudb if it was
parse_args() parses the command-line arguments
handle_crash(error) Generates a crash report.
run(args) Runs the naxxx
main() calls parse_args, set_logger, enable_debugging, and run
