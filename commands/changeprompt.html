The Change Prompt¶

A module to change the prompt variable on a device. Meant to help serial, telnet and other connections that read streams.

The Change Prompt¶

ChangePrompt(adapter[, length, variable, prompt]) Changes a prompt.
ChangePrompt.prompt
return:the alternative prompt to use
ChangePrompt.run()
postcondition:prompt variable is set to new prompt.
