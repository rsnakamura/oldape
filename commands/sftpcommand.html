The SFTP CommandÂ¶

Since the paramiko SSHClient is somewhat buried within the SSHConnection this class was created to hide the need to pull it out and work with it directly. It is a convenience class to create the SFTPClient from the connection and only provides a sub-set of the client commands, the rest if desired would need to be passed to the client itself.

SftpCommand([connection, hostname, ...]) An interface to the SftpCommand

Example Use:

connection = SSHConnection(hostname='192.168.10.24', username='tester')
sftp = SftpCommand(connection=connection)
sftp.get('data.txt', 'remote_data.txt')

TestSftpCommand.test_mkdir() Does the mkdir properly call the sftp mkdir?
TestSftpCommand.test_get() Does the get call the sftp-get?
TestSftpCommand.test_put() Does the put method call the sftp put method?
TestSftpCommand.test_connection_builder() Does the sftp create an SSHConnection if given the right parameters?
TestSftpCommand.test_connection_setter() Does setting the connection re-set the sftp and ssh?
