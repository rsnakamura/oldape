Wget Session¶

The Wget Session repeatedly calls the wget command and monitors the outcome. The initial implementation will be for the BusyboxWget because it is being created for an emergency android-based project.

BusyboxWget Session¶

For the constructor, if only one of max_time or repetitions is given then that will be used to decide when to stop. If both are given, whichever is reached first will cause the session to end.

BusyboxWgetSession(url, connection, storage) A busybox-based wget monitor
BusyboxWgetSession.wget A busy box wget command
BusyboxWgetSession.data_file Name to use for data-file
