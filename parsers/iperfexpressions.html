HumanExpression() The Human Expression matches the human-readable iperf output
HumanExpression.thread_column

HumanExpression.expression

Csv Expression¶

CsvExpression() The Csv Expression holds the expression to match iperf’s csv format
CsvExpression.thread_column
return:the expression to match the thread id
CsvExpression.expression
return:string regular expression to match csv-format
CsvExpression.regex
return:compiled regular expression to match csv-format

Combined Expression¶

CombinedExpression() A Combined expression matches either case (but doesn’t break up the line).
CombinedExpression.expression
rtype:String
CombinedExpression.regex
return:compiled regex that matches both formats
