Serial ConnectionÂ¶

A module to hold a serial connection.

The SerialConnection takes the command-line command as a property and the arguments to the command as parameters.

e.g.

sc = SerialConnection()
output = sc.ls('-l')
print output.output

prints the output of the ls -l command line command
